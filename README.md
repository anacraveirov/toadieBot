# ToadieBot * . ﹢ ˖ ✦

Bot para Discord desenvolvido em Elixir para a disciplida de Proramação Funcional.

## APIs utilizadas

- OpenLibrary API
- Deck Of Cards API
- Exchange API
- Florida Man API
- NagerDate API
- ZenQuotes API

## Comandos

- !livro < nome do livro > < nome do autor >
- !qualquercarta
- !moeda < moeda1 > < moeda2 >
- !feriado <dia/mes/ano>
- !floridaman <dia/mes/ano>
- !frase
- !feriadonaflorida <dia/mes/ano>

## Instalação e Execução

No terminal na pasta do projeto, execute:

```bash
mix deps.get
```

para instalar as dependências do projeto
e depois, execute:

```bash
mix run --no-halt
```

para rodar o projeto!!
